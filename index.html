助詞

Particles

「は」 Topic marker

「が」 Subject marker, Identifier, Conjunction (but)

「も」 Inclusive topic marker (also, too)

「を」 Direct object marker

「に」 Indirect object marker, Destination

「へ」 Direction marker

「で」 Means particle

「と」 Exhaustive list conjunction (and), Preposition (with, as with), (if), Quotation

「や」 Incomplete list marker

「とか」 Incomplete list marker

「の」 Possessive marker, Noun link, Nominalization

「か」 Question marker

「から、まで」(from, until)
